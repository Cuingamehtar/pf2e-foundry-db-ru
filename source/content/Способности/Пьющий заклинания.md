---
title: Пьющий заклинания
---
**Источник:** Pathfinder War of Immortals

- - -

**Требования:** Вашим последним действием был *"Сбор крови"*.

**Частота:** Раз в час

---

Вы временно добавляете в свой репертуар одно заклинание 3-го ранга, определяемое типом существа, чью кровь вы только что выпили (см. таблицу ниже). Если существо обладает более чем одним из перечисленных признаков, выберите один и получите соответствующее заклинание. Если полученное заклинание не входит в список выбранного вами обычая, вы сотворяете его так, как будто оно принадлежит выбранному вами обычаю. Заклинание удаляется из вашего репертуара спустя 24 часа, когда вы в следующий раз используете *"Пьющего заклинания"* или когда вы в следующий раз делаете ежедневные приготовления, в зависимости от того, что произойдёт раньше.

| Признак | Заклинание | Примечания |
| --- | --- | --- |
| @Trait[Aberration]{Аберрация} | [[Заклинания/Вампирское пиршество]] |  |
| @Trait[Fiend]{Бес} | [[Заклинания/Леденящая тьма]] |  |
| @Trait[Giant]{Гигант} | [[Заклинания/Бьющие обломки]] |  |
| @Trait[Humanoid]{Гуманоид} | [[Заклинания/Кризис веры]] |  |
| @Trait[Dragon]{Дракон} | [[Заклинания/Пылающая стрела]] | Вместо урона огнём, это заклинание наносит такой же урон, как дыхание дракона, кровь которого вы выпили (если применимо). |
| @Trait[Animal]{Животное}, @Trait[Beast]{чудовище} | [[Заклинания/Призыв животного]] | Призванные существа появляются как потусторонние версии существ, чью кровь вы выпили. |
| @Trait[Monitor]{Наблюдатель} | [[Заклинания/Надвигающийся рок]] |  |
| @Trait[Celestial]{Небожитель} | [[Заклинания/Священный свет]] |  |
| @Trait[Fey]{Фея} | [[Заклинания/Терновая стена]] |  |

---

**Особенность:** Если у вас есть [[Способности/Бурлящая магия крови]], вы можете добавить это заклинание на 4-м ранге. Если у вас есть [[Способности/Ликующая магия крови]], вы можете добавить это заклинание на 7-м ранге.
